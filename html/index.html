Welcome to Webserv

Introduce

Webserv는 클라이언트 요청을 서버 설정에 맞추어서 응답하도록 설계된 웹서버입니다.
Nginx의 동작을 기반으로 정적 페이지를 제공하고 동적 콘텐츠를 CGI를 통해 처리합니다

Structure

저희가 만든 웹서버는 아래의 구조와 같이 이루어져 있습니다.

  • 매개변수로 들어간 서버 설정 파일을 읽고 ServConf 클래스에 저장합니다.
  • Server 클래스는 ServConf에 저장된 정보를 토대로 서버 소켓을 열고 저장합니다.
  • 소켓의 이벤트를 감지하기 위해 Kernel Queue에 넣어 감지합니다.
  • 서버 소켓의 이벤트를 감지하면 클라이언트 소켓와 연결하여, 연결된 소켓은 Kqueue에 넣습니다.
  • 클라이언트 소켓의 이벤트를 감지하면, 이벤트 필터에 맞춰서 각 필터에 맞는 기능을 수행합니다.
  • 만약 클라이언트가 요청을 보낸다면, 요청 메세지를 임시적으로 보관하여 대기합니다.
  • 응답을 받을 준비가 되었다면, 보관한 메세지를 해석하여 이에 맞는 응답 메세지를 보냅니다.

Start

Webserv를 사용하려면 다음 단계를 따르세요:

  1. 맨 아래 GitHub 링크를 통해 파일을 다운받고, 웹서버를 컴파일합니다.
  2. 각자의 요구 사항에 맞게 구성 파일을 수정하여, 서버를 실행합니다
  3. 웹브라우저 등을 통해 구성 파일 설정에 맞는 서버로 접속합니다
